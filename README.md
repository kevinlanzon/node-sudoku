A test driven command line Sudoku game.

Technologies used
----
- JavaScript
- Node.js
- Jasmine-node
- Travis CI
- Grunt

How to run
----
```sh
$ git clone git@github.com:kevinlanzon/node-sudoku.git
$ cd node-sudoku
$ npm install
```

How to run tests
----
```sh
$ grunt
```
